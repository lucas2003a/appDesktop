>### **Bucle while**
>#### Ejecuta alguna acción **SOLO SI CUMPLE UNA CONDICIÓN**.
>```csharp
>	while(condicion) //Condición a evaluar
>	{
>		// Entra al bucle si cumple la condición
>	}
>```

>### **Bucle do while**
>#### A diferencia del bucle WHILE primero ejecuta una acción y luego evalua alguna condición para continuar con el bucle
>```csharp
>	do
>	{
>		//Acción que se realiza
>
>	}while(condicion); //Condición que se evalua posteriormente
>```
